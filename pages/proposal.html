UI/UX runs through his veins

The design reduces cognitive load by turning institutional text into applicant tasks. The timeline, cards, and call-to-action hierarchy make deadlines and next steps obvious.

  • Mobile-first responsive layout
  • Accessible contrast and skip links
  • Clear journey for PIs and ORIC teams

Maintenance and handover

The repository includes a README, source notes, and production guidance. A real WordPress handover would add editor documentation, update workflows, and a bug-fix support window.

  • Static GitHub demo now
  • WordPress implementation later
  • Guideline document ready to expand

Recommended production stack

WordPress: custom lightweight theme or child theme, Gutenberg blocks, ACF Pro for structured call data, Rank Math/Yoast for SEO governance, caching/CDN, security hardening, analytics, and privacy-first forms.

Integrations: official HEC/RGMS links, Monday.com form embed if needed, downloadable document library, event schema, and admin-editable milestones.
