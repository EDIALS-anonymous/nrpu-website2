# NRPU 2026/27 Website Prototype

A static multi-page concept website for the Higher Education Commission Pakistan National Research Program for Universities (NRPU). Built to demonstrate WordPress development thinking, SEO structure, UI/UX clarity, responsive HTML/CSS/JavaScript, and a maintainable content architecture.

## Pages
- `index.html` — landing page
- `pages/about.html` — program overview
- `pages/eligibility.html` — eligibility checklist
- `pages/apply.html` — application journey and mock intake
- `pages/calendar.html` — 2026/27 call timeline
- `pages/downloads.html` — document library concept
- `pages/contact.html` — applicant support routing
- `pages/proposal.html` — portfolio/proposal proof for WordPress, SEO, UI/UX

## Proposed NRPU 2026/27 timeline
- Advertisement: June 12, 2026
- Last date to apply: July 10, 2026
- Missing/incorrect document notice: July 22, 2026
- Correction deadline: July 30, 2026
- First merit list before external reviewers: August 9, 2026
- Final selected candidates list: September 1, 2026

## GitHub Pages
Upload this folder to a GitHub repository and enable GitHub Pages from the repository settings. The site is pure static HTML/CSS/JS and does not require a build step.
